# 🎲 Dice Roll Statistical Simulation

This project is part of my **2nd Year Data Science Roadmap**. It transitions a basic Python game into a data-gathering tool that records results for statistical analysis.

## 🚀 Overview
Instead of just printing a random number, this script:
1.  **Generates Data:** Simulates 1 or 2 dice rolls based on user input.
2.  **Stores Data:** Uses the **Pandas** library to save every roll into a `.csv` file.
3.  **Analyzes Data:** Automatically calculates the Mean and Mode of your session when you quit.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas (Data Manipulation), Random (Stochastic Simulation)
* **Version Control:** Git & GitHub

## 📂 File Structure
* `dice_roll.py`: The main simulation logic.
* `my_dice_data.csv`: The dataset generated by the script.
* `README.md`: Project documentation.
